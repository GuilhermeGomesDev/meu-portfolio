Relógio Digital

Descrição: Projeto de um relógio digital, que além de mostrar o horário, também informa a data atual e quantos meses, semanas e dias ainda faltam para terminar o ano. Também coloquei a opção de navegação no modo claro ou escuro. Este projeto foi criado com o intuito de exercitar meus conhecimentos em desenvolvimento web.

Tecnologias Usadas: HTML, CSS, JavaScript

Data de Criação: Agosto de 2024
