Lista de Tarefas

Descrição: Projeto de uma lista de tarefas, criada com o intuito de exercitar meus conhecimentos em desenvolvimento web e auxiliar em outros projetos. Ao adicionar uma tarefa, ela ficará abaixo com uma caixa de checagem, que ao ser clicada, riscará a tarefa. Também coloquei a opção de navegação no modo claro ou escuro.

Tecnologias Usadas: HTML, CSS, JavaScript
