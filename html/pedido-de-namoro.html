Pedido de Namoro

Descrição: Projeto de uma página web divertida e interativa que faz a pergunta "Você quer namorar comigo?". A página inclui dois botões: "Sim" e "Não". Toda vez que o usuário clica em "Não", o botão muda de lugar para evitar a resposta negativa. Se o usuário clicar em "Sim", ele é redirecionado para um vídeo especial. Projeto criado com o intuito de exercitar meus conhecimentos em desenvolvimento web.

Tecnologias Usadas: HTML, CSS, JavaScript

Data de Criação: Setembro de 2024
